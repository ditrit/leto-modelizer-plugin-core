ElkLayout

ElkLayout(pluginData, elkParamsopt)

Various tools to arrange the graph view (semi-)automatically. Uses among other things Eclipse Layout Kernel (aka. ELK). ELK's documentation is available at: https://eclipse.dev/elk/ What we use here elkjs, which is a transpiled version from Java to Javascript : https://github.com/kieler/elkjs
