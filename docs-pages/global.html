# NodeData

A component data for automatic layout. Used within ElkLayout. It has tree-related fields : parent, children, depth, in addition to the fields from Component in raw.
Properties:
Name Type Attributes Description
parent NodeData <optional>
 The node's parent.
raw Component <optional>
 The underlying component.
children Array.<NodeData> This node's children.
depth number Distance from the root (root's depth is 0).
